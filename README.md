# Running C Programs in any unix like terminal

```
$ cc helloworld.c
$ ./a.out

or 

$ gcc helloworld.c
$ ./a.out

```
# Most programs are from the book The C Programming Language 
by Brian W. Kernighan and Dennis M. Ritchie
